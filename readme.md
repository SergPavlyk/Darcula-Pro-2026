## Darcula Pro 2026 - Dark theme for Visual Studio 2022 and 2026

**Darcula Pro 2026** is a popular Darcula theme, specially adapted for **Visual Studio 2022 and 2026**.<br>

## 🔤 Recommended Font
For maximum comfort during long working sessions, it is recommended to use a font size of 11–12 pt.
### **Configure** Visual Studio:
    1. Go to Tools → Options
    2. In the left panel, select Environment → More Settings → Fonts and Colors
    3. In the right panel, from the "Show settings for:" dropdown, choose "Text Editor"
    4. In the "Display items:" list, select "Plain Text"
    5. In the "Font (bold type indicates fixed-width fonts):" field, select "Cascadia Mono"
    6. In the "Size:" field, set the value to 11 or 12
    7. Click "OK" to apply the changes
	
## 🎨 Recommended Tab Colorization
For better file navigation, it is recommended to colorize tabs by file extension.
### **Configure** Visual Studio:
    1. Go to Tools → Options
    2. In the left panel, select Environment → Tabs → Document Tabs
    3. In the right panel, find the "Tab colorization method" dropdown
    4. Select "File Extension"
    5. Click "OK" to apply the changes
